GT New Horizons is an open-source community project with over 50 active contributors spanning pack maintenance, custom mod development, quest writing, localization, and more. Every release reflects hundreds of individual contributions: recipe tweaks, quest improvements, bug fixes, and new content. Whether you have a few minutes to file a well-written bug report or weeks to spend writing quest content, there is a place for you in the GTNH contributor community.

## Ways to contribute

**Report bugs** — The GitHub issue tracker is the primary place to report unexpected behavior. Structured issue templates ensure your report includes everything the team needs to investigate. See [Reporting Bugs](/contributing/reporting-bugs) for guidance on filing effective reports.

**Suggest changes** — If something feels unbalanced or could be improved — a recipe tweak, a config adjustment, a balance change — open a "Suggest a Change" issue on GitHub. Community discussion helps shape what gets implemented.

**Quest development** — The questbook has over 3,500 quests and is continuously improved. You can write new quests, improve existing descriptions, fix errors, and add progression guidance. See [Developing Quests](/contributing/quest-development) for the full workflow.

**Quest translation** — The questbook supports multiple languages via lang files. Translators create language-specific files in `config/txloader/load/betterquesting/lang/` and submit them as pull requests. The `template.lang` file is auto-generated and serves as the translation base.

**Suggest new mods** — GTNH has strict criteria for mod additions (open-source license required, must fit the pack's progression philosophy), but well-researched proposals are welcomed via the "Suggest a New Mod" GitHub issue template.

**Code contributions** — GTNH maintains dozens of custom-forked and original mods, all hosted in the [GTNewHorizons GitHub organization](https://github.com/GTNewHorizons). Many of these repos accept pull requests directly.

## Getting started

Join the GTNH Discord at [discord.gg/gtnh](https://discord.gg/gtnh) — it is the primary communication hub for contributors. The `#pack-dev`, `#quest-dev`, and `#mod-dev` channels are where active development discussion happens.

Read the development manual at [gtnh.miraheze.org/wiki/Development](https://gtnh.miraheze.org/wiki/Development) before opening your first pull request. It covers repository structure, tooling expectations, and review conventions.

<Note>
  Larger changes should be discussed in the #pack-dev channel on Discord before opening a PR.
</Note>

All contributions to the modpack config, quest data, and scripts go through the main repository at [github.com/GTNewHorizons/GT-New-Horizons-Modpack](https://github.com/GTNewHorizons/GT-New-Horizons-Modpack).
